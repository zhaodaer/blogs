    # 场景一：天台

    角色：朱长留，安迪，苏心怡，等 7 人

    朱长留（领导者，冷静，有策略） 安迪（技术专家，内向，聪明） 苏心怡（医生，善良，有爱心） 其他六人：各具特色，包括律师，警察，记者，程序员，运动员和艺术家。 朱长留站在天台的边缘，望着下面的城市。他的眼神坚定，声音清晰地说：“我们不能再袖手旁观了。”

    其他人看着他，等待他的下一步计划。朱长留深吸一口气，然后开始讲述他的计划。

    # 场景二：天台会议

    朱长留：“我们的目标是打败财团的不公正行为。我们需要团结起来，用我们的力量对抗他们。”

    安迪：“我可以提供技术支持。我们可以破解他们的财务系统，找出他们的不法行为。”

    苏心怡：“我可以提供医疗援助。我们需要保护我们的队员，防止他们受到伤害。”

    其他人纷纷表示他们愿意提供帮助。他们决定成立一个反财团小队，开始他们的行动。

    # 场景三：反财团小队的行动

    在接下来的剧情中，反财团小队进行了一系列的行动，包括调查、揭露财团的不法行为，以及与财团的对抗。他们遇到了许多困难，但他们始终坚持下去，不断地揭露财团的罪行。

    在一次行动中，他们被财团的人发现，陷入了危险之中。在关键时刻，安迪利用他的技术才能，帮助他们逃脱了困境。

    # 结局：胜利的曙光

    最终，反财团小队成功地揭露了财团的罪行，引起了公众的关注。政府开始对财团进行调查，财团的不法行为终于受到了惩罚。

    在天台，朱长留看着城市的天际线，微笑着说：“我们做到了。” 他转头看向其他的队员，他们的脸上都充满了满足和自豪。

    剧终。
